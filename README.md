# Tool & Technology Stack
- JavaScript (nodeJs)
- Cypress
- VS Code

# To run this repo
1. Clone this repository
```
git clone https://github.com/kasg888/Cypress-testing.git
```
2. Open the folder with VS Code
3. Install npm with
```
npm install
```
4. Install cypress with
```
npm install cypress --save-dev
```
5. Run cypress with
```
npx cypress open
```
or
```
npm run cypress
```
